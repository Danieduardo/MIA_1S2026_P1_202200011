# MIA Proyecto 1 - ExtreamFS
**Carnet:** 20220011  
**Curso:** Manejo e Implementación de Archivos  
**Semestre:** 1S 2026

## Descripción
Aplicación web que simula un sistema de archivos EXT2.

## Tecnologías
- Backend: C++
- Frontend: React
- Reportes: Graphviz

## Ejecución
### Backend
```bash
cd backend
make
./server
```
### Frontend
```bash
cd frontend
npm install
npm run dev
```
